Guia do Usuário

Login e Registro de Usuário

  • Login: Para acessar o aplicativo, use suas credenciais de usuário previamente cadastradas.
  • Registro de Usuário: Se você é novo, clique na opção de registro e preencha os campos necessários para criar uma conta.

Doar uma Informação

  • Doação de Informação: Compartilhe conhecimento ou dados valiosos usando a função de doação. Preencha os campos relevantes e submeta as informações desejadas.

Configurações

  • Configurações: Acesse esta seção para personalizar preferências, notificações, alterar dados, temas, entre outros ajustes disponíveis.

Editar o Perfil

  • Editar Perfil: Atualize suas informações pessoais, como nome, foto de perfil, senha, ou qualquer outro detalhe necessário. Localize a opção de edição dentro do seu perfil.
